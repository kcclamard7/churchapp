Description: The application helps automate the management of the members of the organization, tithes, and offering. The application will also help the treasurer to print invoice, receipt, and others important statement. The software will provide an overview of the entire financial state of the organization.
	Some features:
•	Log into the system and log out securely
•	Create, update and delete user into the system
•	Create, update and delete tithes, offering special contribution
•	Track expenses and display overall accounting balance
•	Generate reports based on a specific data
